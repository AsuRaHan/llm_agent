Добро пожаловать в Smart Hammer Agent!

Ваш локальный AI-ассистент для анализа кода запущен и готов к работе.

Этот интерфейс пока очень прост, но агент активно работает в фоновом режиме.

Информация об API:

Базовый URL API:

Эндпоинт для запросов: POST /api/query

Эндпоинт для статуса: GET /api/status

Эндпоинт для информации о проекте: GET /api/project-info

Эндпоинт для здоровья: GET /api/health

Эндпоинт для конфигурации сервера: GET /api/server-config

Вы можете взаимодействовать с агентом через его REST API.

Для получения дополнительной информации, пожалуйста, обратитесь к документации проекта.
