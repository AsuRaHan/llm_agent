## 🧪 Тестирование

### HTML-страница для тестирования

Для быстрого тестирования без Node.js доступна standalone HTML-страница:

```
E:/my_proj/llm_agent/frontend/doc/test_page.html
```

**Как использовать:**
1. Откройте файл в браузере
2. Выберите вкладку (Чат, WebSocket, Виджеты, Сессии, Ошибки, Консоль)
3. Запустите тесты кнопками
4. Проверьте результаты в блоке "Результаты"

**Подробнее:** [README_TESTING.md](README_TESTING.md)

## 🚀 Быстрый старт

### 1. Тестирование (без Node.js)
```bash
# Просто открыть файл в браузере
file:///E:/my_proj/llm_agent/frontend/doc/test_page.html
```

### 2. Разработка (с Node.js)
```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Тестирование
npm run test
npm run test:e2e
```
