### 1️⃣ Быстрое тестирование (Без Node.js)

**Самый быстрый способ проверить фронтенд:**

```bash
# Просто откройте файл в браузере
E:/my_proj/llm_agent/frontend/doc/test_page.html
```

**Что можно тестировать:**
- ✅ Отправка сообщений
- ✅ WebSocket подключение
- ✅ Виджеты (план, прогресс, ошибки)
- ✅ Сессии
- ✅ Обработка ошибок
- ✅ Консоль логирования

**Как использовать:**
1. Откройте `test_page.html` в браузере
2. Выберите вкладку (Чат, WebSocket, Виджеты...)
3. Нажмите кнопки "Тест: ..."
4. Проверьте результаты

📖 [Полная инструкция](README_TESTING.md)

## 📋 Чеклист первого запуска

### Без Node.js (5 минут)
- [ ] Открыть `test_page.html`
- [ ] Протестировать чат (вкладка "Чат")
- [ ] Протестировать WebSocket (вкладка "WebSocket")
- [ ] Протестировать виджеты (вкладка "Виджеты")
- [ ] Протестировать сессии (вкладка "Сессии")

### С Node.js (15 минут)
- [ ] `npm install`
- [ ] `npm run dev`
- [ ] Открыть http://localhost:3000
- [ ] Протестировать все функции
- [ ] Проверить тесты: `npm run test`
