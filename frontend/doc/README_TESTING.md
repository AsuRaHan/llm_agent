# 🧪 Тестирование Frontend

## 📋 Обзор

HTML-страница для тестирования фронтенда LLM Agent **без Node.js**. Все тесты выполняются в браузере.

## 🎯 Возможности

### ✅ Что можно тестировать:

1. **Чат** - отправка сообщений, рендеринг, авто-скролл
2. **WebSocket** - подключение, отправка, реконнект
3. **Виджеты** - план, прогресс, подтверждение, ошибки
4. **Сессии** - создание, сохранение, загрузка, очистка
5. **Ошибки** - обработка разных типов ошибок
6. **Консоль** - логирование сообщений

## 🚀 Как использовать

### 1. Открыть страницу

```bash
# Просто открыть файл в браузере
file:///E:/my_proj/llm_agent/frontend/doc/test_page.html

# Или через локальный сервер (опционально)
python -m http.server 8000
# Перейти на http://localhost:8000/doc/test_page.html
```

### 2. Пройти тесты

1. Выберите вкладку в верхней части страницы
2. Нажмите кнопки "Тест: ..." для каждого сценария
3. Проверьте результаты в блоке "Результаты"
4. Посмотрите логи в консоли

## 🧪 Примеры тестов

### Тест чата

```javascript
// Тестирование отправки сообщения
testUserMessage();  // ✓ Сообщение пользователя

// Тестирование ответа агента
testAgentMessage();  // ✓ Сообщение агента

// Тестирование системного сообщения
testSystemMessage();  // ✓ Системное сообщение

// Тестирование ошибки
testError();  // ✓ Ошибка

// Тестирование Markdown
testMarkdown();  // ✓ Markdown
```

### Тест WebSocket

```javascript
// Подключение
testConnect();  // ✓ Подключение

// Отправка
testSend();  // ✓ Отправка

// Реконнект
testReconnect();  // ✓ Реконнект
```

### Тест сессий

```javascript
// Создание
testSessionCreate();  // ✓ Создание

// Сохранение
testSessionSave();  // ✓ Сохранение

// Загрузка
testSessionLoad();  // ✓ Загрузка

// Очистка
testSessionClear();  // ✓ Очистка
```

## 🐛 Troubleshooting

### WebSocket не подключается
1. Убедитесь, что сервер запущен на порту 3000
2. Проверьте, что WebSocket сервер доступен
3. Используйте "Симулировать ответ сервера" для тестирования без реального сервера

### Ошибки в консоли
1. Проверьте вкладку "Консоль"
2. Посмотрите на логи с пометкой `[ERROR]`
3. Проверьте настройки CORS на сервере

### Виджеты не отображаются
1. Проверьте, что виджет был создан
2. Нажмите "Очистить" и создайте заново
3. Проверьте консоль на ошибки
