## 🎯 Ключевые возможности

### 🧪 Тестирование (Без Node.js)

**HTML-страница `test_page.html` позволяет:**

1. **Тестировать чат**
   - Отправка сообщений
   - Рендеринг (пользователь, агент, система, ошибка)
   - Markdown
   - Авто-скролл

2. **Тестировать WebSocket**
   - Подключение/отключение
   - Отправка сообщений
   - Реконнект
   - Симуляция ответа сервера

3. **Тестировать виджеты**
   - План действий
   - Прогресс выполнения
   - Подтверждение
   - Ошибки

4. **Тестировать сессии**
   - Создание
   - Сохранение
   - Загрузка
   - Очистка

5. **Тестировать ошибки**
   - Сеть
   - WebSocket
   - Валидация
   - Таймаут

6. **Тестировать консоль**
   - Логирование
   - Очистка

## 🚀 Быстрый старт

### Вариант 1: Без Node.js (5 минут)

```bash
# Просто открыть файл в браузере
file:///E:/my_proj/llm_agent/frontend/doc/test_page.html
```

**Что делать:**
1. Открыть `test_page.html`
2. Пройти тесты по вкладкам
3. Проверить результаты

### Вариант 2: С Node.js (15 минут)

```bash
# Установка
cd E:/my_proj/llm_agent/frontend
npm install

# Запуск
npm run dev

# Тесты
npm run test
```
