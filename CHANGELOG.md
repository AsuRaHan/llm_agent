# CHANGELOG

## [Unreleased] - 2026-05-17

### Добавлено
- **Ручное планирование**: Добавлена кнопка ручного запуска планирования (📋) в интерфейсе чата
- **Редактируемый план**: Пользователь может редактировать шаги плана прямо в интерфейсе (drag-and-drop для перестановки)
- **Визуализация мыслей агента**: Добавлен блок для отображения мыслей агента во время анализа запроса
- **Виджет восстановления после ошибок**: При ошибке выполнения плана пользователь может выбрать стратегию: повторить шаг, пропустить, перепланировать или остановить всё
- **Заморозка FileWatcher**: Добавлена кнопка для приостановки индексации файлов во время выполнения задач
- **Поддержка force_plan**: WebSocket сервер теперь поддерживает явный запрос планирования через параметр `force_plan`
- **Режим планирования**: Добавлены режимы планирования (plan mode, plan mode2)
- **Адаптация ширины чата**: Улучшена адаптация интерфейса чата
- **Улучшение обработки URL**: Улучшена обработка URL в ReadUrlTool
