- **src/FileWatcher.cpp**: Улучшена интеграция с Content Indexer
- **src/WebSocketServer.cpp**: Обновления для поддержки новых функций индексации
- **src/ApiHandlers.cpp/h**: Обновления для новых возможностей
- **src/tools/**: Обновления для CodeSearchTool и ToolManager

### Исправлено

- Исправлены сообщения об ошибках в SessionManager

### Технические детали
- Добавлены новые классы для стратегии разбиения контента (ChunkerStrategy)
- Реализована отдельная логика чтения файлов (FileReader)

- Рефакторинг SessionManager: выделен метод сохранения без блокировки мьютекса для предотвращения двойной блокировки
